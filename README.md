# Kaltura (kaltura)

Kaltura is an open-source video platform providing REST APIs for video management, live streaming, transcoding, player customization, analytics, and virtual classroom integrations. Developers can ingest, process, deliver, and monetize video content at scale across web, mobile, and OTT environments.

**APIs.json:** [https://raw.githubusercontent.com/api-evangelist/kaltura/refs/heads/main/apis.yml](https://raw.githubusercontent.com/api-evangelist/kaltura/refs/heads/main/apis.yml)

## APIs

### Kaltura VPaaS API

The Kaltura Video Platform as a Service (VPaaS) REST API enables developers to manage media assets, perform live streaming, transcode content, deliver video to players, collect analytics, and control access and security for video workflows.

- **Human URL:** [https://developer.kaltura.com/](https://developer.kaltura.com/)
- **Base URL:** `https://www.kaltura.com/api_v3`

### Kaltura OTT Backend API

The Kaltura OTT Backend API provides hundreds of REST-based endpoints giving programmable access to every OTT service including asset management, entitlements, subscriptions, live channels, EPG, and user management for over-the-top video delivery.

- **Human URL:** [https://devdocs.api.ott.kaltura.com/](https://devdocs.api.ott.kaltura.com/)
- **Base URL:** `https://api.ott.kaltura.com`
