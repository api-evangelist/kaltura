# Kaltura GraphQL API

Kaltura is an open-source video platform for education and enterprise. The API covers media entries, playlists, categories, user management, video analytics, live streaming, virtual classroom integrations, Kaltura sessions, and content moderation.

**Endpoint:** No native GraphQL endpoint — conceptual schema derived from Kaltura REST API
**Documentation:** https://developer.kaltura.com/api-docs/

- Reference: https://developer.kaltura.com/api-docs/
